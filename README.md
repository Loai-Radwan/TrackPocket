# TrackPocket

TrackPocket is a Flask web application for tracking personal finances. It allows users to record income and expenses, manage transactions, and view summaries of their financial activity in a simple and organized dashboard.

## Features

- User authentication (sign up / login)
- Add income and expense transactions
- View transaction history
- Categorize transactions
- Basic financial overview/dashboard
- Responsive UI

## Tech Stack

- Python (Flask)
- SQLite (database)
- HTML, CSS, Bootstrap ,JaveScript
- Jinja2 templates
